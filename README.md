# 🎬 NextFrame – Movie Discovery Web Application

NextFrame is a modern movie discovery web application built using HTML, CSS, and JavaScript with OMDb API integration.

## 🚀 Features

* 🔍 Search Movies
* 🎬 Movie Details Popup
* ❤️ Add to Favorites
* 🔔 Toast Notifications
* 🔎 Search Suggestions
* 🎥 Watch Trailer Button
* 🔥 Trending Movies
* 💾 LocalStorage Support
* 📱 Responsive UI
* ✨ Glassmorphism Design

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* OMDb API
